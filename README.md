## How to run.

clone the project.
```
git clone https://github.com/Bigkatoan/Arena4-IsaacSim.git
cd Arena4-IsaacSim
```

create the environment and install dependencies.
```
./create_env
```

run colcon build for start working.
```
./colcon build
```

*NOTE: the installation just long, do not interupt.
